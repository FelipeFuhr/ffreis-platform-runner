# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

To report a security issue, use GitHub's private advisory feature:

**[Report a vulnerability](https://github.com/FelipeFuhr/ffreis-platform-runner/security/advisories/new)**

You will receive a response within 7 days. If the issue is confirmed, a patch will be
released as soon as possible depending on complexity.
